This method returns a single people record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/People/getPeopleBase), please refer to the Swagger
documentation.

## Parameters

| Params | Type     | Required | Description     |
| ------ | -------- | :------: | --------------- |
| id     | `string` |   Yes    | The people `id` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.peopleById('312388');
```
