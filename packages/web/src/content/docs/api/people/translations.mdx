This method returns a single people translation record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/People/getPeopleTranslation), please refer to the Swagger
documentation.

## Parameters

| Params   | Type     | Required | Description           |
| -------- | -------- | :------: | --------------------- |
| id       | `string` |   Yes    | The people `id`       |
| language | `string` |   Yes    | The people `language` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.peopleWithTranslation('312388', 'spa');
```
