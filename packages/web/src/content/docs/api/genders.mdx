This method returns a list of gender records. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Genders/getAllGenders), please refer to the Swagger
documentation.

## Parameters

This method does not require any parameters.

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.genders();
```
