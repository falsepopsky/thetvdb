This method returns a single award category record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Award%20Categories/getAwardCategory), please refer to the Swagger
documentation.

## Parameters

| Params | Type     | Required | Description             |
| ------ | -------- | :------: | ----------------------- |
| id     | `string` |   Yes    | The award category `id` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.awardCategoryById('42');
```
