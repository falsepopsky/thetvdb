This method returns a single season record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Seasons/getSeasonBase), please refer to the Swagger
documentation.

## Parameters

| Params | Type     | Required | Description     |
| ------ | -------- | :------: | --------------- |
| id     | `string` |   Yes    | The season `id` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.seasonById('6365');
```
