This method returns a single extended season record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Seasons/getSeasonExtended), please refer to the Swagger
documentation.

## Parameters

| Params      | Type      | Required | Description                                     |
| ----------- | --------- | :------: | ----------------------------------------------- |
| id          | `string`  |   Yes    | The season `id`                                 |
| translation | `boolean` | Optional | Include translation information in the response |

## Examples

### Without translations

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.seasonByIdExtended('6365');
```

import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.seasonByIdExtended('6365', true);
