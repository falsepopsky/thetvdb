This method returns a single season translation record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Seasons/getSeasonTranslation), please refer to the Swagger
documentation.

## Parameters

| Params   | Type     | Required | Description           |
| -------- | -------- | :------: | --------------------- |
| id       | `string` |   Yes    | The season `id`       |
| language | `string` |   Yes    | The season `language` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.seasonWithTranslation('6365', 'rus');
```
