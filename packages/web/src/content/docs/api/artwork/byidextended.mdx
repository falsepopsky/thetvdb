<Tag apiPath={'/v4/artwork/:id/extended'} />

This method returns a single extended artwork record. For official documentation of the [endpoint](https://thetvdb.github.io/v4-api/#/Artwork/getArtworkExtended), please refer to the Swagger documentation.

## Parameters

| params | type     | Required | Description      |
| ------ | -------- | :------: | ---------------- |
| id     | `string` |   Yes    | The artwork `id` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.artworkByIdExtended('63237874');
```
