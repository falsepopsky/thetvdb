<Tag apiPath={'/v4/artwork/:id'} />

This method returns a single artwork record. For official documentation of the [endpoint](https://thetvdb.github.io/v4-api/#/Artwork/getArtworkBase), please refer to the Swagger documentation.

## Parameters

| Params | Type     | Required | Description      |
| ------ | -------- | :------: | ---------------- |
| id     | `string` |   Yes    | The artwork `id` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.artworkById('63237874');
```
