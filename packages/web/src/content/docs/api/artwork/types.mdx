This method returns a list of artwork status records. For official documentation of the [endpoint](https://thetvdb.github.io/v4-api/#/Artwork%20Types/getAllArtworkTypes), please refer to the Swagger documentation.

## Parameters

This method does not require any parameters.

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.artworkTypes();
```
