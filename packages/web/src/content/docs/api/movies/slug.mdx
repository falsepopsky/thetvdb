This method returns a single movie record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Movies/getMovieBaseBySlug), please refer to the Swagger
documentation.

## Parameters

| Params | Type     | Required | Description      |
| ------ | -------- | :------: | ---------------- |
| slug   | `string` |   Yes    | The movie `slug` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.movieBySlug('macross-do-you-remember-love');
```
