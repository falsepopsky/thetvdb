This method returns a single movie record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Movies/getMovieBase), please refer to the Swagger
documentation.

## Parameters

| Params | Type     | Required | Description    |
| ------ | -------- | :------: | -------------- |
| id     | `string` |   Yes    | The movie `id` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.movieById('12586');
```
