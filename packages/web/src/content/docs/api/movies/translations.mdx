<Tag apiPath={'/v4/movies/:id/translations/:language'} />

This method returns a single movie translation record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Movies/getMovieTranslation), please refer to the Swagger
documentation.

## Parameters

| Params   | Type     | Required | Description          |
| -------- | -------- | :------: | -------------------- |
| id       | `string` |   Yes    | The movie `id`       |
| language | `string` |   Yes    | The movie `language` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.movieWithTranslation('12586', 'spa');
```
