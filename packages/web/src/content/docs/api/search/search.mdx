This method returns a list of query search records. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Search/getSearchResults), please refer to the Swagger
documentation.

## Parameters

| params                  | type     | Required | Description                                    |
| ----------------------- | -------- | :------: | ---------------------------------------------- |
| queryParams             | `object` |   Yes    | Search parameters to restrict the result       |
| queryParams.company     | `string` | Optional | Restrict results to a company                  |
| queryParams.country     | `string` | Optional | Restrict results to a country                  |
| queryParams.director    | `string` | Optional | Restrict results to a director                 |
| queryParams.language    | `string` | Optional | Restrict results to a language                 |
| queryParams.limit       | `string` | Optional | Limit the list of records                      |
| queryParams.network     | `string` | Optional | Restrict results to a specific network         |
| queryParams.offset      | `string` | Optional | Offset results                                 |
| queryParams.page        | `string` | Optional | Restrict results to a specific page            |
| queryParams.primaryType | `string` | Optional | Restrict results to a specific type of company |
| queryParams.query       | `string` |   Yes    | Restrict results to a specific title           |
| queryParams.remote_id   | `string` | Optional | Restrict results to a specific remote `id`     |
| queryParams.type        | `string` | Optional | Restrict results to a type                     |
| queryParams.year        | `string` | Optional | Restrict results to a year                     |

## Examples

### With required queryParams parameters

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.search({ query: 'saint seiya' });
```

```js {4}
const client = new TheTVDB('access token');
const data = await client.search({ query: 'yu gi oh', type: 'series', limit: '1' });
```
