This method returns a list of entity records. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Search/getSearchResultsByRemoteId), please refer to the Swagger
documentation.

## Parameters

| params | type     | Required | Description     |
| ------ | -------- | :------: | --------------- |
| id     | `string` |   Yes    | The remote `id` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.searchRemoteId('42444');
```
