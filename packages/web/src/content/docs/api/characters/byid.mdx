<Tag apiPath={'/v4/characters/:id'} />

This method returns a single character record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Characters/getCharacterBase), please refer to the Swagger
documentation.

## Parameters

| params | type     | Required | Description        |
| ------ | -------- | :------: | ------------------ |
| id     | `string` |   Yes    | The character `id` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.characterById('64140522');
```
