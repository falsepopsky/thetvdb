This method returns a single episode translation record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Episodes/getEpisodeTranslation), please refer to the Swagger
documentation.

## Parameters

| params   | type     | Required | Description            |
| -------- | -------- | :------: | ---------------------- |
| id       | `string` |   Yes    | The episode `id`       |
| language | `string` |   Yes    | The episode `language` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.episodeWithTranslation('40', 'spa');
```
