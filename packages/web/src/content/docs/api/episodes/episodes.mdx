This method returns a list of episode records. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Episodes/getAllEpisodes), please refer to the Swagger
documentation.

## Parameters

| params | type     | Required | Description                         |
| ------ | -------- | :------: | ----------------------------------- |
| page   | `string` | Optional | Restrict results to a specific page |

## Examples

### Without parameters

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.episodes();
```
