This method returns a single list record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Lists/getListBySlug), please refer to the Swagger
documentation.

## Parameters

| Params | Type     | Required | Description     |
| ------ | -------- | :------: | --------------- |
| slug   | `string` |   Yes    | The list `slug` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.listBySlug('1001');
```
