This method returns a single list translation record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Lists/getListTranslation), please refer to the Swagger
documentation.

## Parameters

| Params   | Type     | Required | Description         |
| -------- | -------- | :------: | ------------------- |
| id       | `string` |   Yes    | The list `id`       |
| language | `string` |   Yes    | The list `language` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.listWithTranslation('17', 'spa');
```
