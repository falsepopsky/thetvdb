<Tag apiPath={'/v4/content/ratings'} />

This method returns a list of content ratings records. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Content%20Ratings/getAllContentRatings), please refer to the Swagger
documentation.

## Parameters

This method does not require any parameters.

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.contentRatings();
```
