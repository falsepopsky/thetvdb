This method returns a single company record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Companies/getCompany), please refer to the Swagger
documentation.

## Parameters

| Params | Type     | Required | Description      |
| ------ | -------- | :------: | ---------------- |
| id     | `string` |   Yes    | The company `id` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.companyById('4');
```
