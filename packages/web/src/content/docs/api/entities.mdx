This method returns a list of entity types records. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Entity%20Types/getEntityTypes), please refer to the Swagger
documentation.

## Parameters

This method does not require any parameters.

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.entities();
```
