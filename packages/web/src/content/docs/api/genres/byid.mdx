This method returns a single genre record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Genres/getGenreBase), please refer to the Swagger
documentation.

## Parameters

| params | type     | Required | Description    |
| ------ | -------- | :------: | -------------- |
| id     | `string` |   Yes    | The genre `id` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.genreById('27');
```
