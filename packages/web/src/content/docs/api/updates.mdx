This method returns a list of updated records. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Updates/updates), please refer to the Swagger
documentation.

## Parameters

| params             | type     | Required | Description                                                  |
| ------------------ | -------- | :------: | ------------------------------------------------------------ |
| queryParams        | `object` |   Yes    | Search parameters to restrict the result                     |
| queryParams.since  | `string` |   Yes    | The timestamp to get updates in epoch Unix format in seconds |
| queryParams.type   | `string` | Optional | Restrict results to a specific type of entities              |
| queryParams.action | `string` | Optional | Restrict results to a specific type of action                |
| queryParams.page   | `string` | Optional | Restrict results to a specific page                          |

## Examples

### With required queryParams parameters

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.updates({ since: '1696118400' });
```
