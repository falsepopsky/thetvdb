<Tag apiPath={'/v4/inspiration/types'} />

This method returns a list of inspiration types records. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/InspirationTypes/getAllInspirationTypes), please refer to the Swagger
documentation.

## Parameters

This method does not require any parameters.

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('your token');
const data = await client.inspirationTypes();
```
