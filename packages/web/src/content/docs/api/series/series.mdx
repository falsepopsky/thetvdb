This method returns a list of series records. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Series/getAllSeries), please refer to the Swagger
documentation.

## Parameters

| Params | Type     | Required | Description                         |
| ------ | -------- | :------: | ----------------------------------- |
| page   | `string` | Optional | Restrict results to a specific page |

## Examples

### Without parameters

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.series();
```
