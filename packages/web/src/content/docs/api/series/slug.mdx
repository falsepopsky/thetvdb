This method returns a single serie record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Series/getSeriesBaseBySlug), please refer to the Swagger
documentation.

## Parameters

| Params | Type     | Required | Description      |
| ------ | -------- | :------: | ---------------- |
| slug   | `string` |   Yes    | The serie `slug` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.serieBySlug('yu-gi-oh');
```
