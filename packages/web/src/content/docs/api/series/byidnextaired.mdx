This method returns a single serie record. For official documentation of the
[endpoint](https://thetvdb.github.io/v4-api/#/Series/getSeriesNextAired), please refer to the Swagger
documentation.

## Parameters

| Params | Type     | Required | Description    |
| ------ | -------- | :------: | -------------- |
| id     | `string` |   Yes    | The serie `id` |

## Example

```js {4}
import { TheTVDB } from '@untidy/thetvdb';

const client = new TheTVDB('access token');
const data = await client.serieByIdNextAired('78878');
```
